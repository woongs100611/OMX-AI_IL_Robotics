OMX 로봇 팔을 활용한 모방학습(Imitation Learning) 프로젝트입니다.

---

## 1. 모방학습 개요

| 항목 | 내용 |
|------|------|
| **정의** | 사람의 동작을 보고 로봇이 따라 배우는 학습 방식 |
| **특징** | 시행착오 없이 사람의 시범 데이터로 바로 학습 |
| **장점** | 빠른 학습, 직관적인 데이터 수집 |
| **활용** | 로봇 팔 조작, 물체 집기, 반복 작업 자동화 |

### 학습 흐름

| 단계 | 내용 |
|------|------|
| 1단계 | 사람이 OMX를 직접 조종하여 동작 시범 |
| 2단계 | 관절 각도 및 움직임 데이터 저장 |
| 3단계 | AI 모델이 데이터 패턴 학습 |
| 4단계 | 로봇이 학습된 동작 스스로 실행 |

## 2. About OMX - 서보모터

| 항목 | 내용 |
|------|------|
| **서보모터란** | 각도와 위치를 정밀하게 제어할 수 있는 모터 |
