## 1. 모방학습 개요

### 모방학습이란?

모방학습(Imitation Learning)은 사람이 직접 시범을 보여주면
로봇이 그 동작 데이터를 학습하여 스스로 따라하는 AI 학습 방법입니다.
강화학습과 달리 수천 번의 시행착오 없이 사람의 데이터만으로 학습할 수 있습니다.

### 다른 학습 방식과 비교

| 학습 방식 | 설명 | 단점 |
|-----------|------|------|
| **지도학습** | 정답 데이터를 주고 학습 | 대량의 라벨 데이터 필요 |
| **강화학습** | 보상을 주며 시행착오로 학습 | 학습 시간이 매우 오래 걸림 |
| **모방학습** | 사람의 시범 동작을 보고 학습 | 시범 데이터의 질에 의존 |

### 모방학습 종류

| 종류 | 설명 | 특징 |
|------|------|------|
| **BC (Behavior Cloning)** | 시범 데이터를 그대로 따라 학습 | 구현 간단, 오차 누적 문제 있음 |
| **DAgger** | 학습 중 추가 데이터를 계속 수집 | BC보다 안정적, 사람 개입 필요 |
| **GAIL** | 시범과 비슷하게 행동하도록 학습 | 높은 성능, 구현 복잡 |

### 학습 흐름

| 단계 | 작업 | 설명 |
|------|------|------|
| 1단계 | 데이터 수집 | 사람이 OMX를 직접 조종하며 동작 시범 |
| 2단계 | 데이터 저장 | 관절 각도, 속도, 위치 정보를 기록 |
| 3단계 | 전처리 | 노이즈 제거, 데이터 정규화 |
| 4단계 | 모델 학습 | AI가 시범 데이터의 패턴을 학습 |
| 5단계 | 검증 | 학습된 모델의 동작 정확도 테스트 |
| 6단계 | 배포 | OMX에 모델 적용 후 자율 동작 실행 |

## 2. About OMX - 서보모터

### 서보모터란?

서보모터(Servo Motor)는 단순히 회전만 하는 일반 모터와 달리
각도와 위치를 정밀하게 제어할 수 있는 모터입니다.
내부에 위치 센서(엔코더)가 있어 현재 각도를 피드백으로 받아
원하는 각도로 정확하게 움직일 수 있습니다.

### 서보모터 종류

| 종류 | 설명 | 사용 예시 |
|------|------|-----------|
| **RC 서보모터** | 0~180도 범위 제어, PWM 신호 사용 | 소형 로봇, 드론 |
| **디지털 서보모터** | 빠른 응답속도, 높은 정밀도 | 산업용 로봇 |
| **다이나믹셀** | 각도/속도/토크 모두 제어 가능 | OMX 로봇 팔 |

### OMX 로봇 팔 구성

| 관절 | 서보모터 역할 | 동작 범위 |
|------|--------------|-----------|
| **베이스** | 좌우 회전 | 0 ~ 360도 |
| **숄더** | 앞뒤 상하 움직임 | 0 ~ 180도 |
| **엘보우** | 팔꿈치 굽힘/펴기 | 0 ~ 180도 |
| **리스트** | 손목 회전 | 0 ~ 180도 |
| **그리퍼** | 물체 잡기/놓기 | 열림/닫힘 |

### 서보모터 제어 방식

| 항목 | 내용 |
|------|------|
| **신호 방식** | PWM (Pulse Width Modulation) |
| **제어 주기** | 20ms (50Hz) |
| **각도 제어** | 펄스 폭으로 각도 결정 (1ms=0도, 2ms=180도) |
| **통신 방식** | UART / TTL / RS-485 |

### 모방학습에서 서보모터의 역할

| 단계 | 서보모터 역할 |
|------|--------------|
| **데이터 수집 시** | 사람이 움직이는 각 관절의 각도값을 실시간 기록 |
| **학습 시** | 기록된 각도 데이터가 AI 학습의 입력값으로 사용 |
| **실행 시** | AI가 예측한 각도값대로 서보모터가 정밀하게 구동 |
